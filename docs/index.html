Canva Linux

A community-maintained open source desktop wrapper for Canva on Linux. Native performance, persistent sessions, and seamless integration.

View on GitHub Installation Guide

Native Feel

Optimized Electron wrapper with custom styling for a true desktop experience.

Flatpak Support

Easy installation and sandboxed security via Flathub-ready manifests.

Wayland Ready

Full support for modern Wayland sessions with X11 fallback if needed.

EyeDropper

Includes a custom color picker flow integrated directly into the Canva editor.

Quick Start

Install the latest development version via Flatpak:

git clone https://github.com/coletivo420/canva-linux.git
cd canva-linux
./canva-linux.sh --install
