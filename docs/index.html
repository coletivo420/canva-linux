Canva Linux

A community-maintained desktop wrapper for using Canva on Linux with persistent sessions, internal tabs, OAuth handling, custom EyeDropper support, and desktop integration.

Persistent Sessions

Stay logged in across restarts without repeating the sign-in flow.

Internal Tabs

Manage multiple designs within a single window while keeping the shell focused.

OAuth Support

Secure login flows work through Google, Facebook and other supported providers.

Custom EyeDropper

Integrated color picking works in Linux desktop sessions with the project's custom flow.

Wayland & X11

Modern display-server support keeps the wrapper usable across common Linux setups.

GPU Acceleration

High-performance rendering is enabled by default for a smoother desktop experience.
